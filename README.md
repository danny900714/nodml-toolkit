# 公文管理系統 工具包

臺南市政府第3代公文管理系統工具包，提供許多自動輸入及實用功能。

## 功能

- 在郵件登錄畫面中，於備註中輸入收件者姓名，即可自動選擇收件單位
- 在收文登錄畫面中，自動取消勾選「保留輸入值」選項
- 在收文登記簿畫面中，自動選取收文類別、收文人員和分頁選項，以便快速列印
